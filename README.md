## Тестовое задание
Это тестовое задание по Unity/Mirror.

Необходимо реализовать сервис отправки сетевых сообщений (NetworkMessages)
между сервером и клиентом с использованием плагина Mirror.

В плагине уже реализован механизм обмена сетевыми сообщениями,
однако он имеет ограничение: сервер отключает клиента, если клиент не
зарегистрировал обработчик для конкретного типа сообщения. Требуется
разработать собственное решение, не модифицируя исходный код плагина,
на основе стандартных сетевых сообщений Mirror, которое позволит
отправлять сообщения только тем клиентам, которые явно подписались на
соответствующий тип сообщения.

Сценарий работы должен быть следующим:
1. Запускается сервер (или хост);
2.  Подключается клиент;
3.  Клиент подписывается на получение сообщения типа HelloMessage;
4.  Сервер получает информацию о подписке клиента;
5.  Сервер отправляет клиенту сообщение HelloMessage с текстом "Hello Client!";
6.  Клиент принимает сообщение и выводит текст в консоль.

Технологии:
- Unity `2022.3.62f3`
- Mirror
- Zenject (DI)

## Запуск и проверка
Открыть проект в Unity `2022.3.62f3`, перейти по на сцену `Boot`, включить `Play Mode`, 
нажать на кнопку `Host (Server + Client)`. В консоли должно появиться приветственное сообщение.
